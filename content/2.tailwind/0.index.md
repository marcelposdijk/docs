# TailwindCss

## Getting started

There is a module for nuxt3 that we can use:

::code-group

```bash [npm]
npm install @nuxtjs/tailwindcss
```

```bash [yarn]
yarn install @nuxtjs/tailwindcss
```

::

Besides that we can install the followin tailwind plugins that could make our lives a litte bit easier:

```bash
npm install @tailwindcss/forms @tailwindcss/typography
```

We change nuxt.config.ts by adding @nuxtjs.tailwindcss as module:

```ts
export default defineNuxtConfig({
  modules: ['@nuxtjs/tailwindcss']
}
```
